# Contributing to mloda

We welcome contributions from the community to help us improve and expand mloda. Whether you're interested in developing plugins, fixing bugs, or adding new features, your input is invaluable.

## Ways to Contribute

### Core Development
For contributions to the mloda core framework, see our [Development Guide](https://mloda-ai.github.io/mloda/development/).

### Plugin Development
The easiest way to extend mloda is by creating plugins:

- **[mloda-registry](https://github.com/mloda-ai/mloda-registry)** - Browse 40+ guides covering the complete plugin development journey
- **[mloda-plugin-template](https://github.com/mloda-ai/mloda-plugin-template)** - Use our cookiecutter template to create standalone plugin repositories with pre-configured CI/CD

### Report Issues
Found a bug or have a feature request? [Open an issue](https://github.com/mloda-ai/mloda/issues/).
